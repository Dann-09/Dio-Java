# Anotações Java

Iniciar projeto java com arquivo ".java"

o código deve iniciar em "public class minha class {}"

Se a minha classe for executavel ela precisa ter um metodo principal (especial) chamado "main"

Void - executar sem retornar nada

System - classe com operaçoes de input e output

Toda classe precisa estar dentro de SRC

Todo nome composto deve conter as inicias maiusculas

Todo arquivo ".Java" deve começar com nome maiusculo 

A Classe deve possuir o mesmo nome do arquivo ".java"

Toda variavel deve ser com letras minusculas

Se a variavel tiver nome composto a primeira letra da segunda palavra deve ser maiuscula

ex: ano
anoFabricacao

o nome dessa pratica chama camelCase

Para quando a variavel nao for ter o valor alterado deve ser colocado com "Final" antes da variavel

ex: final string BR = 'Brasil' e a variavel deve ser toda maiuscula 

Para declarar uma variável nós nao podermos utilizar caracteres numeros e simbolos
Existem regras especifica

Regra de declaracao de váriavel:

Tipo NomeVariavel = Atribuição ()

Declarando metodos 

TipoRetorno NomeObjetivoNoInfinitivo Parametro(s)

Java beans - iniciativa conceitual da comunidade para que a gente possa ter uma estruturação de escrita (melhor forma de expressar os códigos)
